410 /* The DFSDM channels are defined as follows:
411  - in 16-bit LSB the channel mask is set
412  - in 16-bit MSB the channel number is set
413  e.g. for channel 3 definition:
414  - the channel mask is 0x00000008U (bit 3 is set)
415  - the channel number 3 is 0x00030000
416  --> Consequently, channel 3 definition is 0x00000008U | 0x00030000 = 0x00030008 */
